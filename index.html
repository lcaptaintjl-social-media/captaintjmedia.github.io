Captain TJ Media

Automated social media publishing for the Captain TJ YouTube channel — a Great Lakes maritime content creator documenting life aboard massive commercial vessels.

About This Application

Captain TJ Media is a personal automation tool that posts video content to connected social media accounts (YouTube, TikTok, Instagram, Facebook) on behalf of the channel owner. It does not collect user data, does not serve third parties, and is not available to the general public.

TikTok Integration

This application uses the TikTok Content Posting API to upload short-form videos to TikTok as drafts. Videos are submitted using the video.upload scope and appear in the creator's inbox for review before publishing. No user data beyond the OAuth access token is accessed or stored.
